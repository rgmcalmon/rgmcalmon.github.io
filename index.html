Summary: Latitude vs. X

The aim of this project was to compare latitude against four variables — temperature, cloudiness, humidity, and windspeed — and to see what sorts of relations exist between them. The data displayed on this site was all taken at a specific time on April 6th, 2019. The dataset was formed by first selecting points on earth uniformly at random using numpy's random.uniform function, and then calling citypy to find cities close to the points. Since at least 500 cities was desired and citypy only returns cities for points sufficiently near to a city, 1500 points had to be chosen initially. The OpenWeatherMap API was then called to pull weather data on each of the cities.

Having assembled the dataset, matplotlib was used to make scatterplots of the four comparisons. One of the four comparisons exhibited a strong unimodal tendency, so a quadratic regression was computed to estimate the center of the distribution, as well as to add a helpful curve to overlay over the scatterplot. The other three comparisons exhibited no strong relations, so they were left as-is.
